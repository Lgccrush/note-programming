### Linux 命令

```sh 
tar -zxvf #解压
mv 文件名 文件夹 #移动文件到某个文件夹下面
set number #vim下显示行号
lsof -i:端口 #查找端口号
/关键词 #vim查找
nohup java -jar xxx.jar & #后台运行
```

#### 压缩

```sh
tar -zcvf 
```
