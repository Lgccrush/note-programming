pom.xml引入jar

```xml
				<!--字符串表达式运算-->
        <dependency>
            <groupId>com.googlecode.aviator</groupId>
            <artifactId>aviator</artifactId>
            <version>3.0.1</version>
        </dependency>
```

使用

```java
  public static void main(String[] args) {
        String str = "1+2";
        System.out.println(AviatorEvaluator.execute(str));
  }
```
