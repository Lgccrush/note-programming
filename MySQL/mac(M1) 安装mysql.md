### 安装

直接点击下载的安装包正常安装即可

### 配置

```bash
打开终端 
vim ~/.bash_profile  #编辑配置文件
添加 export PATH=$PATH:/usr/local/mysql/bin
source ~/.bash_profile #刷新配置文件
mysql -u root -p  #登录mysql
```
